====== eUtilities ======
Insieme di programmi di utilità. Trova file duplicati (o mancanti) nelle directory. Rimuovere i file duplicati in una directory. Convertire DFM Delphi da binarie a testuali (e viceversa). Scaricare un insieme di URL. Rinominare tutti i file in una directory con il group_sequenceNumber schema, ...

===== Descrizione =====
^ Nome ^ Lingua ^ Descrizione ^
^ConvertForm | {{ico:eng.png|ENG}} |Convertitore di file DFM Delphi dal formato binario a quello testuale e vice versa |
^Find        | {{ico:eng.png|ENG}} |Cerca file duplicati (o mancanti) all'interno di due directories |
^HTTPGet     | {{ico:eng.png|ENG}} |HTTPGet scarica un insieme di URL |
^Purge       | {{ico:eng.png|ENG}} |Rimuove i file duplicati all'interno di un alberatura di directory |
^Rename      | {{ico:eng.png|ENG}} |Rinomina tutti i file di una directory secondo lo schema IDGruppo_numeroSequenza, dove gruppo è l'insieme dei file che che iniziano con "IDGruppo_" |
