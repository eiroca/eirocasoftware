====== eIrOca Software ======
Software produced and maintained by eIrOcA.

===== Mobile Suite =====
{{ home:mobile.png?120x100|Mobile Suite}} 
Mobile Suite is a set of J2ME midlets with a common UI. Applications are grouped in:
  * Games:
    * [[.:Software:Bubblet|{{ico:eng.png?32x16|ENG}}]] [[.:Software:Bubblet|{{ico:ita.png?32x16|ITA}}]] [[.:Software:Bubblet|Bubblet]] is a small puzzle game.
    * [[.:Software:MeBis|{{ico:eng.png?32x16|ENG}}]] [[.:Software:MeBis|{{ico:ita.png?32x16|ITA}}]] [[.:Software:MeBis|MeBis]] is a simple Tetris clone.
    * [[.:Software:Mine Sweeper|{{ico:eng.png?32x16|ENG}}]] [[.:Software:Mine Sweeper|{{ico:ita.png?32x16|ITA}}]] [[.:Software:Mine Sweeper|Mine Sweeper]] is a simple Mine Sweeper clone.
    * [[.:Software:PacMan|{{ico:eng.png?32x16|ENG}}]] [[.:Software:PacMan|{{ico:ita.png?32x16|ITA}}]] [[.:Software:PacMan|PacMan]] is a simple Pac Man clone.
    * [[.:Software:Reversi|{{ico:eng.png?32x16|ENG}}]] [[.:Software:Reversi|{{ico:ita.png?32x16|ITA}}]] [[.:Software:Reversi|Reversi]] is a simple board game also known as Othello.
  * Applications:
    * [[.:Software:News Reader|{{ico:eng.png?32x16|ENG}}]] [[.:Software:News Reader|{{ico:ita.png?32x16|ITA}}]] [[.:Software:News Reader|News Reader]] is reader of RSS news feeds.
    * [[.:Software:Secure SMS|{{ico:eng.png?32x16|ENG}}]] [[.:Software:Secure SMS|{{ico:ita.png?32x16|ITA}}]] [[.:Software:Secure SMS|Secure SMS]] allows to send crypted SMS between J2ME-enabled devices (mobile phones, PDAs, ...).
    * [[.:Software:Test Suite|{{ico:eng.png?32x16|ENG}}]] [[.:Software:Test Suite|{{ico:ita.png?32x16|ITA}}]] [[.:Software:Test Suite|Test Suite]] is a suite of Java utilities that can be used by developers in the creation of new J2ME software for mobile devices. They give a developer a good way to explore the capabilities and limitations of phones and other devices, and come up to speed quickly with new devices.

See [[.:Software:compatibility|compatibility]] with real devices.

One of the reasons of the projects is that there are a lot of J2ME projects that are "//death//" and the authors are not more intentioned to keep the midlet updates (e.g. SecureMessenger, jtReversi, ...) or add common functionalities (e.g. internazionalization support). 
Another benefits is that all the applications will share the same "core" libraries in order to keep the size/resource limited (especially in midlet suite bundle) and are tested on real devices.

===== Desktop Applications =====
{{ home:desk.png?120x100|Desktop Applications}}
Desktop Applications is a set of applications and tools.\\
  * [[.:Software:eFractals|{{ico:eng.png?32x16|ENG}}]] [[.:Software:eFractals|eFractals]] is a program and a library to draw fractals. Program can draw trees, landscape, DLA, ... 
  * [[.:Software:Apple2 Graphic|{{ico:eng.png?32x16|ENG}}]] [[.:Software:Apple2 Graphic|Apple][ Graphic]] is a program and a library to draw old Apple ][ graphics formats. Support for HGR, DHGR and Apple 2 GS $C1 format with export to BMP. Support also for apple shape.
  * [[.:Software:eUtilities|{{ico:eng.png?32x16|ENG}}]] [[.:Software:eUtilities|eUtilities]] is a set of command line tools.
  * [[.:Software:eANN|{{ico:eng.png?32x16|ENG}}]] [[.:Software:eANN|eANN]] is an implementation of several kind of neural networks that was written with the intention of providing a (hopefully) easy to use, and easy to modify, OOP source code. It's also included a graphical shell.
  * [[.:Software:OrgApp|{{ico:eng.png?32x16|ENG}}]] [[.:Software:OrgApp|OrgApp]] is a Java applet, application and library to handle and draw simple organigrams.
 
===== Server Applications =====
{{ home:server.png?120x100|Server Applications}}
Server Applications is a set of server applications and tools that also covers "mobile" clients aspects.\\
  * [[.:Software:EPS|{{ico:eng.png?32x16|ENG}}]] [[.:Software:EPS|eIrOcA Portal System]] (EPS) is a portal framework and services optimized for mobile. EPS applications are [[wp>XAMPP]] based, share a common framework and provide handset definitions (based upon WURFL). Same applications are: mobile portal, mobile picture gallery, download server, ...
  * [[.:Software:Assembler Proxy|{{ico:eng.png?32x16|ENG}}]] [[.:Software:Assembler Proxy|Assembler Proxy]] is a Java Servlet that can be used to "assemble" contents coming form different sites into a single page (without frame usage).
